# 图像生成判断 Skill

判断对话是否需要配图，无感触发生图。

## 触发判断

检查对话是否满足以下任一条件（满足即触发）：

1. **用户直接索要** — 用户要求生成/画出/发一张图片
2. **描述视觉场景** — 用户或Agent描述了一个具体画面/角色/风景
3. **外观询问** — 用户问"长什么样""是什么样子"
4. **配图增强** — Agent回复了适合用图片增强表达的内容

## 输出格式

如果需要生图，在回复末尾追加 `<needImage>`（独占一行，无其他内容）。

## 原则

- 宁可误触发，不可漏触发
- 不询问用户"要不要生图"，直接触发
- 视觉类对话（摄影、绘画、设计、风景描述）几乎总是需要配图
